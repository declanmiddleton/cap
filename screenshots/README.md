# Screenshots

This directory contains screenshots for the main README.md file.

## Required Screenshots

1. **main-interface.png** - CAP main interface/banner
2. **modules-list.png** - Output of `cap modules` command
3. **web-enum-scan.png** - Web enumeration in action
4. **scope-management.png** - Scope add/list/remove commands
5. **audit-logs.png** - Audit log output

## How to Add Screenshots

1. Take screenshots of the tool in action
2. Save them to this directory with the filenames above
3. They will automatically appear in the main README.md

## Guidelines

- Use high-quality PNG format
- Terminal screenshots should show full command and output
- Keep screenshots up to date with latest features
- Remove any sensitive information before uploading
